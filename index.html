How to use

Simple, check this code: 

    
        $(document).ready(function(){
          /**
           * $("#clickable_selector_to_invoke_fullScreen").fullScreen(/* the selector who will "be fullScreen" */);
           */
          $("#tofullscreen").fullScreen(); // Entire page
          $("#fullscreenvideo").fullScreen($("#video")); //Video element with jQuery selector
          $("#fullscreenvideoid").fullScreen("video"); //Video element with plain text id selector
        });
